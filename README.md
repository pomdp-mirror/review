# Research Paper Website Template

Simple Vite and React research paper website template styled after the neighboring Eureka project page.

## Customize

- Edit paper content in `src/App.jsx`.
- Edit page styling in `src/App.css` and shared layout styles in `src/index.css`.
- Replace `src/assets/hero.png` with your own teaser figure or image.
- Update the page title and description in `index.html`.

## Commands

```sh
npm run dev
npm run build
npm run lint
```

No extra node modules are required beyond the existing project dependencies.
